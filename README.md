# COLLEGE-HEALTH-CARE
ABOUT THE PROJECT
----------------------------
 Developed an online Vit health care portal to help my college students get online prescriptions
from doctors and check availability of medicines in medical store .

----------------------------

----------------------------

TECHNOLOGIES USED
----------------------------

- Python 
- Django 
- SQLLite
- Bootstrap ( HTML / CSS / Javascript )
